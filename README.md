# Patili-Melekler

High-conversion, mobile-first static website for **Patili Melekler Pet Shop & Pet Kuafor**.

## Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Features
- Premium visual hero and curated gallery from local assets
- Responsive desktop + mobile layout
- Mobile navigation menu
- Social proof + services + pricing + reviews + contact sections
- Placeholder lead form (UI only, backend not connected)

## Local preview
Open `index.html` directly in a browser, or run a local static server.
